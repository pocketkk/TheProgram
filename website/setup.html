üîß Troubleshooting

"Invalid API key" error

Make sure you copied the full key with no extra spaces. Anthropic keys start with sk-ant-.

"Insufficient credits" error

You need to add billing to your Anthropic account. Free trial credits may have expired.

Gemini images not generating

Make sure you're using a Gemini API key (not a different Google API). Check that you haven't hit rate limits.
